# Python Mastery Path

Welcome to the ultimate Python learning resource. We have curated the best open-source materials into a structured path to take you from beginner to master.

## Learning Path

### [1. Beginner](./1.beginner)
Start here if you are new to Python. Covers the basics: variables, loops, functions, and object-oriented programming.
*Based on the amazing "30 Days of Python".*

### [2. Mid (Intermediate)](./2.mid)
Deepen your knowledge with intermediate topics, notebooks, and practical exercises.
*Includes interactive notebooks.*

### [3. Pre-Advance](./3.pre-advance)
Learn best practices and Design Patterns. Write clean, maintainable, and pythonic code.
*Focus on Design Patterns.*

### [4. Advance](./4.advance)
Dive into algorithms, data structures, and the internal workings of Python.
*Code snippets and advanced concepts.*

### [5. Master](./5.master)
Specialized topics like Data Science. Master the tools used by professionals.
*Based on the "Python Data Science Handbook".*
